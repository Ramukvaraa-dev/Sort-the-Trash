♻️ Eco Garbage Shooter

Sort falling trash and compete for the highest score.

🎮 Play Game

🎮 How To Play

Select the correct trash type (Plastic, Paper, or Organic) using the panel on the left, then click anywhere on screen to shoot at matching falling garbage. Right match = +10 pts. Wrong = −5 pts and combo reset.

🏆 Leaderboard

Scores are saved in your browser — no server needed. Enter a username (3–12 chars, letters / numbers / underscores) and hit Save Score to appear on the board.
