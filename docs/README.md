# Eco Garbage Shooter 🌍🔫

A harder, smoother, GitHub Pages-ready arcade shooter.

## Features
- Tougher scaling
- Better UI/UX
- Smooth animations
- Combo system
- Power-ups
- Hardcore progression
- Mobile support
- GitHub-ready repo setup

## Setup
Open `index.html` in a browser.

## GitHub Pages
Enable Pages from:
Settings → Pages → Deploy from branch → main → /root
